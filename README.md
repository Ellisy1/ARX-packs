# Аркс 3
Чтобы создать игру, постройте на далеких координатах неломаемую комнату, снизу который должны быть блоки `minecraft:deny`. 
В ней должен быть мировой спавн `/setworldspawn` и должна быть сущность `arx:lobby_character_creation`
Координаты спавна игрока после регистрации можно поменять в файле `ARX NAP BP/functions/tp/2_spawn.mcfunction`
